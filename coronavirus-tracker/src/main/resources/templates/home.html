About This Project

his project displays the cases per county in the United States, the total number of cases to that date in each county, and new cases in each county. Additionally, at the top, there is a count for the total number of cases in the United States and new cases reported in the last day. When the project is deployed to elasticbeanstalk, the webapp should refresh with new data every day.
This project was meant to improve my skills in Java, Spring Boot, and Server Applications. For future updates, I hope to put data visualizations and host on elasticbeanstalk as well as maybe some more fancy front end features. Additionally, I might want to change my data source to a more dependable data source like a a restapi.
